PHP - Ejercicios básicos (1,2,3,4,5,7,8-19)
=================================================

Cómo ejecutar rápido (servidor embebido de PHP)
-----------------------------------------------
1) Instala PHP (>= 8.x). Comprueba en terminal:
   php -v

2) Arranca un servidor local desde este directorio:
   php -S localhost:8000 -t .

3) Abre en el navegador:
   http://localhost:8000/index.php

VS Code (opcional)
------------------
- Abre la carpeta en VS Code (File > Open Folder...).
- Extensión recomendada: "PHP Intelephense".
- Puedes usar el terminal integrado para ejecutar el servidor:
  php -S localhost:8000 -t .

Apache/XAMPP (alternativa)
--------------------------
- Copia esta carpeta dentro de htdocs y accede desde:
  http://localhost/php_curso_basico/index.php
